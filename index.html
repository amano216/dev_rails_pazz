🐳 Docker コマンド

コンテナ一覧表示

docker ps

実行中のコンテナを表示

すべてのコンテナ表示

docker ps -a

停止中も含めてすべて表示

イメージ一覧

docker images

ダウンロード済みのイメージを表示

コンテナ起動

docker start [コンテナ名]

停止中のコンテナを起動

コンテナ停止

docker stop [コンテナ名]

実行中のコンテナを停止

コンテナ削除

docker rm [コンテナ名]

コンテナを削除

イメージ削除

docker rmi [イメージ名]

イメージを削除

コンテナ内でコマンド実行

docker exec -it [コンテナ名] bash

コンテナ内に入る

Docker Compose起動

docker-compose up

複数コンテナを一括起動

Docker Compose停止

docker-compose down

複数コンテナを一括停止

Docker Composeでbundle install

docker compose run web bundle install

コンテナ内でGemをインストール

Docker ComposeでSorceryインストール

docker compose run web rails g sorcery:install

Sorcery認証機能をセットアップ

Docker Compose再起動

docker compose restart

全コンテナを再起動

Docker Composeで開発サーバー起動

docker compose exec web bin/dev

コンテナ内で開発サーバーを起動

Docker Composeでマイグレーション

docker compose exec web rails db:migrate

コンテナ内でDBマイグレーション実行

Docker Composeでマイグレーション状態確認

docker compose exec web rails db:migrate:status

マイグレーションの適用状態を確認

Docker Composeでコントローラー生成

docker compose exec web rails g controller users

usersコントローラーを生成

Docker Composeでセッションコントローラー生成

docker compose exec web rails g controller user_sessions

user_sessionsコントローラーを生成

💎 Gem コマンド

Gemインストール

gem install [gem名]

特定のGemをインストール

Gemアンインストール

gem uninstall [gem名]

Gemを削除

インストール済み一覧

gem list

インストール済みGemの一覧

Gemの更新

gem update [gem名]

特定のGemを最新版に更新

Bundlerでインストール

bundle install

Gemfileに記載のGemをインストール

Bundle更新

bundle update

すべてのGemを更新

Gemの情報表示

gem info [gem名]

Gemの詳細情報を表示

Gemを検索

gem search [キーワード]

リポジトリからGemを検索

🐧 Linux コマンド

ディレクトリ内容表示

ls

現在のディレクトリのファイル一覧

詳細表示

ls -la

隠しファイルを含む詳細情報

ディレクトリ移動

cd [ディレクトリ名]

指定ディレクトリへ移動

現在地表示

pwd

現在のディレクトリパスを表示

ディレクトリ作成

mkdir [ディレクトリ名]

新しいディレクトリを作成

ファイル削除

rm [ファイル名]

ファイルを削除

ディレクトリ削除

rm -rf [ディレクトリ名]

ディレクトリを強制削除

ファイルコピー

cp [元] [先]

ファイルをコピー

ファイル移動

mv [元] [先]

ファイルを移動または名前変更

ファイル内容表示

cat [ファイル名]

ファイルの内容を表示

ファイル検索

find . -name "[ファイル名]"

ファイルを名前で検索

テキスト検索

grep "[検索文字]" [ファイル]

ファイル内のテキストを検索

権限変更

chmod 755 [ファイル名]

ファイルの権限を変更

プロセス表示

ps aux

実行中のプロセス一覧

📦 Git コマンド

リポジトリ初期化

git init

新しいGitリポジトリを作成

クローン

git clone [URL]

リモートリポジトリをコピー

ステータス確認

git status

変更状況を確認

ファイル追加

git add .

すべての変更をステージング

コミット

git commit -m "メッセージ"

変更を記録

プッシュ

git push origin main

リモートに送信

プル

git pull

リモートから取得

ブランチ一覧

git branch

ブランチの一覧表示

ブランチ作成・切替

git checkout -b [ブランチ名]

新しいブランチを作成して切替

マージ

git merge [ブランチ名]

ブランチを統合

ログ表示

git log --oneline

コミット履歴を一行表示

差分表示

git diff

変更内容を表示

取り消し

git reset --hard HEAD

すべての変更を取り消し

スタッシュ

git stash

変更を一時保存

Issue一覧

ワークフロー実行

gh workflow run [名前]

ルート確認

テスト実行

アセットプリコンパイル
